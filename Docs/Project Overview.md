# LF Robot 2026

My first serious embedded systems project.

Goal: Build a competition-ready line follower robot while learning embedded systems fundamentals.
# Engineering Design Document (EDD)

## 1. Project Objective

Build a reliable, high-speed Line Follower Robot from scratch to learn embedded systems engineering and participate in Mindspark 2026.

---

## 2. Learning Objectives

- Learn STM32 programming
- Learn Embedded C
- Understand IR sensors
- Learn PWM motor control
- Learn PID control
- Learn hardware debugging
- Learn Git and documentation

---

## 3. Functional Requirements

The robot should:

- Follow a black line on a white surface
- Handle sharp turns
- Complete the track reliably
- Be battery powered
- Be lightweight
- Be easy to debug

---

## 4. Constraints

- Student budget
- Limited development time
- Components should be easily available in India
- Must be repairable

---

## 5. Success Criteria

- Robot completes the Mindspark track consistently.
- All hardware is documented.
- All code is stored on GitHub.
- Every design decision is documented.
